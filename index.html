About This Project

This tool automatically creates weekly playlists featuring the latest releases from artists you follow on Spotify, plus similar artists you might enjoy.

Features

  • Scans your followed artists for new releases
  • Discovers similar artists and their latest tracks
  • Creates weekly playlists automatically
  • Filters for releases from the past 7 days
  • Secure authentication with Spotify API
